Komponenta 6/7

Ta podvojena komponenta vsebuje HTML kodo, ki ni prikazana, saj vsebuje style="display: none". Še vedno pa je ta prisotna v DOM strukturi in se v ozadju tudi posodablja.

Ura s posodabljanjem na 1000ms: {{ currentDate.getHours() }} : {{ currentDate.getMinutes() }} : {{ currentDate.getSeconds() }}

Ura s posodabljanjem na 200ms: {{ currentDate2.getHours() }} : {{ currentDate2.getMinutes() }} : {{ currentDate2.getSeconds() }}
